This is my CISC482 IA08 assignment: doodleCanvas, developed with Android Studio and JetPack Compose.

The main goal of doodleCanvas is to enable the user to draw on their phones.

Key features:
  1. Drawable canva
  2. Tool Panel with 8 different brush colors(default is black) and adjustable brush size(1px to 50px, default is 10px)
  3. Clear button to clear the drawings
  4. **New feature: Undo button to undo the last drawn stroke
